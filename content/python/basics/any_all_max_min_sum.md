## Create Data


```python
data = [34, 54, 50, 20, 20]
```

## any()


```python
# Return true if any element is True
any(data)
```




    True



## all()


```python
# Return true if all elements are True
all(data)
```




    True



## max()


```python
# Return max value
max(data)
```




    54



## min()


```python
# Return the min value
min(data)
```




    20



## sum()


```python
# Return the total value
sum(data)
```




    178
